# Images Directory

This directory should contain the following images:

## Required Images

- **hero.jpg** - Hero section image
  - Recommended size: 1200x800px or larger
  - Format: JPG, WebP, or PNG
  - Should be a professional headshot or coaching-related image

- **profile.jpg** - Profile/About page image
  - Recommended size: 600x600px or larger
  - Format: JPG, WebP, or PNG
  - Professional headshot of Alejandro Valencia

## Adding Images

1. Add your images to this directory
2. Ensure they are optimized for web (compressed, appropriate format)
3. Update the image paths in components if using different filenames
